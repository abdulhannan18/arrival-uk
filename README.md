Arrival UK

Arrival UK is a Swift-based iOS app project designed to help international students in the UK understand important first steps after arrival.

The app focuses on practical setup topics such as SIM cards, banking, GP registration, National Insurance, local transport, groceries, housing, jobs, and student life.

Why this project exists

International students often receive important information from many scattered places: university emails, induction sessions, friends, websites, social media, and personal experience. Arrival UK aims to organize this information into a simple mobile app experience so new students can find useful guidance more easily.

Main features

* iOS app structure built with Swift
* Category-based student guidance
* Local setup information for new international students
* Content stored in structured JSON
* Firebase/backend planning included
* Documentation and project notes included
* Designed to scale into more categories over time

Tech stack

* Swift
* Xcode
* JSON content structure
* Firebase/backend planning
* iOS app architecture
